pub enum Value {
    U64(u64),
    I64(i64),
    Bytes(Vec<u8>),
    String(String),
    Array(Vec<Value>),
    Object(BTreeMap<ObjectKey, Value>),
    F64(f64),
    Bool(bool),
    Null,
}

An enum over all possible CBOR types.

Variants

U64(u64)

Represents an unsigned integer.

Null

Represents the absence of a value or the value undefined.
